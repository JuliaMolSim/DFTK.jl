# Contributions
Contributions to the code in any form is very welcome,
just [submit a pull request](https://github.com/JuliaMolSim/DFTK.jl/pulls)
on github. If you want to contribute but are unsure where to start, take a look
at the list of issues tagged [good first issue](https://github.com/JuliaMolSim/DFTK.jl/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)
(relatively easy tasks suitable for newcomers) or [help wanted](https://github.com/JuliaMolSim/DFTK.jl/issues?q=is%3Aissue+is%3Aopen+label%3A%22help+wanted%22)
(more sizeable but well-defined and isolated).
Don't hesitate to ask for help, through github,
email, [JuliaMolSim zulip][chat-url] or [Matrix chat][matrix-url].

# AI policy

You are very welcome to submit pull requests with AI help, with the following conditions:

- You need to disclose what the AI did and what you did, and indicate the AI model used.

- You need to actually provide added value to the PR (this doesn't need to be in the form of actual code, but can be research, testing, design, etc). You need to fully understand the code and be able to demonstrate that it is working as intended. If anybody could have prompted an AI to generate your PR, it is probably not worth submitting.

- Using AI in discussions without acknowledgement (making it write your posts) is unacceptable. Even if your English is broken, you're not sure of what you're doing, etc, we'd much rather talk to you than your AI.
